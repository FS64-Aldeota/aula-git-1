git init (Inicia o git no repositório local)

git add . (Adiciona todos os arquivos modificados para dentro do commit )

git commit -m "mensagem" (Sela o commit passando uma mensagem como identificador)

git remote add origin https... (Adiciona um remote, ou seja, ele aponta para onde vai ser enviado no GitHub. )

git push origin Fulano (Envia o código para a branch Fulano)

git branch (Lista as branches e indica qual branch vocês estão)

git checkout -b fulano (Cria uma branch e muda para ela)

git checkout main (Muda para a  branch main)

git pull origin main (Atualiza a branch main, baixa o código que está no github)
